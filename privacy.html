Política de Privacidade – Assistente de Ocorrências Disciplinares IFMT Este assistente é utilizado exclusivamente para registro de ocorrências disciplinares no âmbito institucional do IFMT. Dados coletados: - Nome do discente - Turma - Disciplina - Descrição da ocorrência - Nome do servidor responsável Finalidade: Os dados são utilizados exclusivamente para envio de ocorrências à coordenação e supervisão pedagógica. Armazenamento: Os dados podem ser armazenados em sistema interno para fins de registro institucional. Compartilhamento: Os dados são compartilhados apenas com: - Coordenação de curso - Supervisão pedagógica Segurança: Medidas técnicas são adotadas para proteção dos dados. Contato: Para dúvidas, procure a coordenação do IFMT. Última atualização: 2026
